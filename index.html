Walkman


Como ejercicio se escogió la marca Walkman y se diseño una identidad totalmente diferente para ella. Nos preguntamos: ¿Cómo sería la marca si lanzara al mercado una nueva edición? y trabajamos este caso hipotético, creando un nuevo logo, seleccionando una nueva paleta de color, tipografías, etc.




Edición especial


Después de este trabajo, se decidió continuar experimentando con el lanzamiento de una edición especial. El proposito de esta sería jugar con la nostalgía que dicha marca aporta retomando patrones y elementos gráficos de los 80s o principios de los 90s.



  • Fecha: Septiembre 2018



Six Flags

Otra marca que se seleccionó con el proposito de rediseñar fue el parque de diversiones Six Flags. Se diseño una nueva identidad a partir de nuevos recursos gráficos y también se creó una serie de íconos para acompañarla.

  • Fecha: Octubre 2018



Bilab 83

Para este ejercicio se creó la idea de un laboratorio multidisciplinario ficticio que convinaría ciencia y arte. Se desarrolló una detallada identidad gráfica para dicho lugar.

  • Fecha: Diciembre 2018
